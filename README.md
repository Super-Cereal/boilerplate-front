# vtb-hack-workwolf

Проект команды workwolf на VTB API hackathon 2024

## Как запустить

- Установить зависимости из корня репозитория (устанавливаем из корня, чтобы активировать pre-commit хук)
  ```bash
  npm run deps
  ```
- Запускаем бэкенд
  ```bash
  npm run backend:dev
  ```
- Запускаем фронтенд
  ```bash
  npm run frontend:dev
  ```
